# Stock Market Analytics Dashboard
## Overview
This project is a Stock Market Analytics Dashboard developed using Python, Machine Learning, and Power BI. The objective is to analyze historical stock market data, identify trends, measure volatility, and build a predictive model for stock price analysis.
The project demonstrates the complete data analytics workflow, including data collection, preprocessing, feature engineering, machine learning, and business intelligence visualization.
## Features
Analyzes historical stock price movements.
Visualizes long-term market trends.
Moving Average Analysis
20-Day Moving Average (MA20)
50-Day Moving Average (MA50)
Identifies bullish and bearish market signals.
Volatility Analysis
Calculates daily returns.
Measures annualized volatility.
Assesses market risk and stability.
Predictive Analytics
Uses Linear Regression for stock price prediction.
Employs lag-based feature engineering.
Evaluates model performance using standard metrics.
Interactive Power BI Dashboard
Historical stock performance visualization.
Moving average comparison.
Volatility monitoring.
Actual vs Predicted price comparison.
KPI cards for model performance metrics.
